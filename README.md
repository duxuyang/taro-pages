
# taro 小程序文件夹和路由自动生成，组件自动生成,自动根据js或者ts版本区分
## 选中page文件夹，右击鼠标 选择（创建page文件 taro） 根据输入的文件名称创建页面，自动添加路径
## 在components目录，右击鼠标，选择（创建page文件 taro）根据输入的文件名称创建组件

## 支持 vue2 、vue3、react 组件与页面模版
## react 创建hooks模板
